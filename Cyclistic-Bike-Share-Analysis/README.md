# Cyclistic Bike-Share Usage Analysis

## Business Questions
1. How do annual members and casual riders use bikes differently?
2. Why would casual riders buy annual memberships?
3. How can Cyclistic use digital media to influence casual riders to become members?

## Dataset
- Cyclistic historical bike-share data (Q1 2020)
- Source: Divvy Bikes
- Format: CSV

## Data Cleaning
-> Removed trips with invalid or unrealistic durations
->Calculated trip duration in minutes
-> Filtered outliers (>120 minutes)
-> Extracted hour, day, and month from timestamps

## Analysis & Visualizations
->Average trip duration by user type
-> Total rides by user type
->Hourly usage patterns
-> Weekly and monthly behavior
->Top start stations

## Key Insights
->Casual riders take longer, leisure-oriented trips
-> Members ride more frequently during commute hours
-> Casual riders show strong weekend usage

## Recommendations
->Target casual riders with weekend-focused membership promotions
-> Use digital media to highlight cost savings
-> Focus campaigns around high-demand stations
